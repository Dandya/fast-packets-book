# fast-packets-book

***

## Описание

На момент написания этой работы в России разрабатывается параллельно множество продуктов по анализу, обработке и передаче сетевого трафика при скорости свыше 10 Гб/с. Основой таких продуктов служат системы по захвату сетевого трафика. В работе вы найдёте описание, разбор модулей ядра Linux и готовые примеры наиболее популярных из них.

Целью работы является передача собственного опыта работы с системами захвата сетевого трафика и упрощение разработки тем, кто только начинает изучать эту сферу.

## Содержание

1. Основы работы сетевого стека ядра Linux
2. Создание стенда для запуска примеров
3. AF_PACKET
4. AF_XDP
5. Netmap
6. PF_RING
7. DPDK
8. Перспективы развития
