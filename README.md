# fast-packets-book

***

## Описание

На момент написания этой работы в России разрабатывается параллельно множество продуктов по анализу, обработке и передаче сетевого трафика при скорости свыше 10 Гб/с. Основой таких продуктов служат системы по захвату сетевого трафика. В работе вы найдёте описание, разбор модулей ядра Linux и готовые примеры наиболее популярных из них, а также попробуете создать собственную систему захвата сетевого трафика.

Целью работы является передача собственного опыта работы с системами захвата сетевого трафика и упрощение разработки тем, кто только начинает изучать эту сферу.

## Содержание

1. [Основы работы сетевого стека ядра Linux](Linux-network-stack.md)
2. Создание стенда для тестирования систем захвата сетевого трафика
3. Cоздание системы захвата сетевого трафика
4. AF_PACKET
5. AF_XDP
6. Netmap
7. PF_RING
8. DPDK
9. С
10. Заключение
