# fast-packets-book

***

## Описание

На момент написания этой работы в России разрабатывается параллельно множество продуктов по анализу, обработке и передаче сетевого трафика при скорости свыше 10 Гб/с. Основой таких продуктов служат системы по захвату сетевого трафика. Здесь вы найдёте описание, разбор устройства и готовые примеры наиболее популярных из них.

Целью работы является передача опыта работы с системами захвата сетевого трафика и упрощение разработки тем, кто только начинает изучать эту сферу.

## Необходимые для понимания знания

Понимание языка программирования «C», умение устанавливать операционные системы на базе ядра «Linux» и работать в терминале.
