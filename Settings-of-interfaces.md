## Настройка с помощью параметров драйвера

Параметры работы драйвера задаются при вызове программ `modprobe` или `insmod` после названия модуля. Чтобы узнать список доступных параметров, нужно воспользоваться программой `modinfo`, которая выводит информацию о модуле, включая его параметры.

```sh
# Пример вывода параметров модуля igb
modinfo ./src/igb.ko | grep "parm"
```

```bash
# Пример запуска модуля igb c распределением пакетов по двум очередям RSS
insmod ./src/igb.ko RSS=2
```
